Example 1: Basic Datepicker initially set to Date Object

Raw Value: {{myDate}}

Value With Medium Date Filter: {{myDate | date:'medium'}}

Code:
$scope.myDate = Date.parse("10/26/2013");
<aa-datepicker ng-model='myDate' hover-text='This is the hover text' placeholder='Click to Set Date' icon-class='icon-time' />

Example 2: Basic Datepicker with placeholder initially set to Null

Raw Value: {{myDate}}

Value With Medium Date Filter: {{myDate | date:'medium'}}

Notice that the button initially reads "Click to Set Date". This comes from the placeholder attribute since the model is null.

Code: 

$scope.myDate = null;
<aa-datepicker ng-model='myDate' hover-text='This is the hover text' placeholder='Click to Set Date' icon-class='icon-time' />
